01 UBUNTU İNFAZI VE DONANIM KALKANLARI

Otonom sistemin çökmelerine neden olan "Güç Tasarrufu", "Uyku Modu" ve gereksiz masaüstü yazılımları (bloatware) kökünden sökülüp atılır.

1.1. Bloatware (Çöp Yazılım) Temizliği
sudo apt-get update && sudo apt-get upgrade -y
sudo apt-get purge -y libreoffice* thunderbird* transmission* rhythmbox* remmina* gnome-sudoku mahjongg
sudo apt-get autoremove -y && sudo apt-get clean
1.2. USB Uyku İptali (Mikrofon ve Radar ASLA Uyumayacak)
# Extlinux boot dosyasını nano editör ile açın:
sudo nano /boot/extlinux/extlinux.conf

# Dosyanın içindeki "APPEND" ile başlayan uzun satırın EN SONUNA bir boşluk bırakıp şunu ekleyin:
usbcore.autosuspend=-1

# (CTRL+O ile kaydedin, ENTER'a basın, CTRL+X ile çıkın)
1.3. Wi-Fi Kopma Engelleyici (Güç Koruması İptali)
# Wi-Fi güç korumasını 3 (Açık) değerinden 2 (Kapalı) değerine çekiyoruz:
sudo sed -i 's/wifi.powersave = 3/wifi.powersave = 2/' /etc/NetworkManager/conf.d/default-wifi-powersave-on.conf
sudo systemctl restart NetworkManager
1.4. Maksimum Performans (MAXN Modu) ve Otonom Fan
sudo nvpmodel -m 0
sudo jetson_clocks

05 ZİHİN VE ALGI MODELLERİNİN SSD'YE HAPSEDİLMESİ (Ön-Yükleme)

Sistemin internet koptuğunda sağır ve dilsiz kalmaması için, tüm zihin parçalarını Air-Gapped (Çevrimdışı) çalışacak şekilde SSD'ye indiriyoruz. Bu işlem arka plan çöküntülerini tamamen engeller.

5.1. Qwen 2.5 (Ana Zihin) ve YOLOv8 (Şahin Göz) Dosyaları
cd ~/alice_os/storage/models/

# Ana Zihin (Yaklaşık 2.18 GB)
wget -c https://huggingface.co/Qwen/Qwen2.5-3B-Instruct-GGUF/resolve/main/qwen2.5-3b-instruct-q4_k_m.gguf

# Şahin Gözü (YOLO)
wget -c https://github.com/ultralytics/assets/releases/download/v8.2.0/yolov8n.pt -O yolov8n.pt
5.2. Görsel Zihin, Ses DNA ve TTS Modelleri Otonom İndiricisi
# Bu komutları Terminale tek tek yapıştırın. Sistem tüm modelleri bulup HF Cache ve Config dosyalarına kazıyacaktır.
# (Emin olmak için source ~/alice_env/bin/activate komutunu girmiş olun)

# 1. Moondream2 (Görsel Zihin - Fotoğraf Analizi ve Mutfak Şefi için)
python3 -c "from transformers import AutoModelForCausalLM, AutoTokenizer; AutoModelForCausalLM.from_pretrained('vikhyatk/moondream2', revision='2024-08-26', trust_remote_code=True); AutoTokenizer.from_pretrained('vikhyatk/moondream2', revision='2024-08-26')"

# 2. SpeechBrain ECAPA-TDNN (25MB Ses DNA'sı - Aryen ve Rana'yı ayırmak için)
python3 -c "from speechbrain.inference.speaker import EncoderClassifier; EncoderClassifier.from_hparams(source='speechbrain/spkrec-ecapa-voxceleb', savedir='/home/$USER/alice_os/storage/models/ecapa')"

# 3. Faster-Whisper (Kulak - Konuşmayı Metne Döker)
python3 -c "from faster_whisper import WhisperModel; WhisperModel('small', device='cpu', compute_type='int8')"

# 4. Silero VAD (İnsan Sesi Radarı - Halüsinasyon Engelleyici)
python3 -c "import torch; torch.hub.load(repo_or_dir='snakers4/silero-vad', model='silero_vad', trust_repo=True)"

# 5. OpenWakeWord (Pusu Uyandırıcısı - "Alis" Sesi Algılama)
python3 -c "import openwakeword; openwakeword.utils.download_models()"

# 6. Coqui TTS (Kusursuz Türkçe Kadın Sesi Ağzı)
python3 -c "from TTS.api import TTS; TTS(model_name='tts_models/tr/f_boun/vits')"

06 OTONOM UYANIŞ (SYSTEMD DİRİLİŞ SERVİSİ)

Elektrikler gelip Jetson açıldığı an, şifre girmeye veya terminal komutu yazmaya gerek kalmadan Edge OS arayüzünün tam ekran açılmasını sağlayan çekirdek kodudur.

6.1. Servisi Oluşturma
sudo nano /etc/systemd/system/alice.service

Aşağıdaki kodu açılan siyah ekrana yapıştırın.
(DİKKAT: USER_ADINIZ yazan 4 yeri, Ubuntu'yu kurarken belirlediğiniz gerçek kullanıcı adınızla değiştirin!)

6.2. Systemd Mühür Kodları
[Unit]
Description=Alice Edge AI Operating System
After=network.target graphical.target sound.target display-manager.service
Wants=display-manager.service

[Service]
Type=simple
User=USER_ADINIZ
Environment="DISPLAY=:0"
Environment="XAUTHORITY=/home/USER_ADINIZ/.Xauthority"
Environment="PULSE_RUNTIME_PATH=/run/user/1000/pulse"
WorkingDirectory=/home/USER_ADINIZ/alice_os
ExecStart=/home/USER_ADINIZ/alice_env/bin/python main.py
Restart=always
RestartSec=10
LimitNOFILE=65536

[Install]
WantedBy=graphical.target
6.3. Servisi Kaydetme ve Ateşleme
# Dosyayı kaydedip çıktıktan sonra (CTRL+O, Enter, CTRL+X)
sudo systemctl daemon-reload
sudo systemctl enable alice.service

# Operasyonel Kurulum Tamamlandı. İnfaz işlemini başlatın:
sudo reboot
